# Maiumixmaster

Website for **Maiumixmaster** — a mixing &amp; mastering service built exclusively around commercial dance music (house, tech house, bass and beyond).

## Stack

Single-page static site. No build step — plain HTML/CSS in [`index.html`](index.html), with the hero image in `assets/`.

## Run locally

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy

Hosted on Vercel. `index.html` sits at the repo root, so it serves at the domain root with zero config — every push to `main` auto-deploys.
